# DevJobs UI

A modern and responsive job board UI built using React and CSS, featuring dynamic job cards, clean layouts, hover animations, and reusable components inspired by real-world hiring platforms.

---

## ✨ Features

- ⚛️ Built with React + Vite
- 🎨 Clean and modern UI design
- 📱 Responsive card-based layout using CSS Grid
- 🏢 Dynamic company/job card rendering
- 🔖 Interactive save button with Lucide React icons
- ✨ Smooth hover animations and transitions
- 🖼️ Company logo integration
- 💼 Job tags like Full-Time, Senior Level, Remote, etc.
- 🚀 Reusable component structure

---

## 🛠️ Technologies Used

- React.js
- CSS3
- Vite
- Lucide React Icons

---

## 🎯 What I Learned

- Working with React components and props
- Rendering dynamic data using arrays
- Using CSS Grid and Flexbox for layouts
- Adding hover animations and transitions
- Importing and using external icon libraries
- Creating modern card-based UI layouts
